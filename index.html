The Pantry Geniemore_vert Website Repository

Inventory Management Appmore_vert Website Repository
Inventory Manaagement Appclose

Web app for a local business to streamline their steel yard’s inventory management. This application allows a worker to select the material they are either picking up or dropping off from a 60 acre lot. Once the item is selected they are directed on a Google Map with pins of where the item is located or should go. Once that task is performed the employee can send a auto populated email with what their task was and select a manager to send the email to. Utilized HTML, Bulma, CSS, and jQuery to create dynamic webpages that are easy to use and read. Used the Google Maps api as well as SMTPjs.com to email the inventory department.
